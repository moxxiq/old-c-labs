Вибрає номер завдання лабораторної

labNumber - номер лабораторної
Цикл do while використовується церез необхідність спочатку ввести дані, а потім перевірити належність відрізку [1;2]
Для перевірки введення цілого десяткового числа використовується функція inputD()
